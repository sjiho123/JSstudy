Web worker란?
web worker를 이해하기전에 스레드의 개념부터 알아볼 필요가 있다

스레드란.

프로세스내에서 실제로 작업을 수행하는 주체를 의미한다.
모든 프로세스에는 한 개 이상의 스레드가 존재하여 작업을 수행한다
두개 이상이면 멀티스레드 프로세스라고 한다.

프로세스란.

사용자가 작성한 프로그램이 운영체제에 의해 메모리 공간을 할당받아 실행 중인 것으로
단순히 실행중인 프로그램이라고 할 수 있다.
이러한 프로세스는 프로그램에 사용되는 데이터와 메모리등의 자원 그리고 스레드로 구성이된다



web worker
MDN 에서는 다음과 같이 정의되어 있다.
Web Worker는 스크립트 실행을 메인쓰레드가 아니라 백그라운드 스레드에서 실행할 수 있도록 해주는 기술 입니다.
이기술을 통해 무거은 작업을 분리된 스레드에서 처리할 수 잇으며
이를통해 메인 스레드 는 멈춤 속도저하없이 동작할 수 있게됩니다.
